============ sdfkm help ============
Support CMD:
help            : Show command list. 

egensm2         : KMT EKM GEN SM2 KEY
                <prog> ekm egensm2 keyindex [num] pin [str] [sigk|enck]
                         default keyindex 1
                         default pin 11111111
                         default sigk

./gdacmmktool  ekm egensm2 keyindex 1 pin 11111111 enck
./gdacmmktool  ekm egensm2 keyindex 2 pin 11111111 enck

./gdacmmktool  ekm egensm2 keyindex 1 pin 11111111 sigk
